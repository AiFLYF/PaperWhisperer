What it does

让每篇论文都无处可藏

01 / Summary

结构化文档摘要

自动提炼论文核心观点，生成覆盖背景、方法、结论的结构化摘要，快速把握全文脉络。

02 / Citations

关键引用提取

智能识别适合直接引用或做笔记的关键片段，标注来源位置，一键整理参考资料。

03 / Mindmap

Mermaid 可视化结构图

生成论文逻辑层级与知识结构的 Mermaid 图表，支持 SVG 导出，便于展示与分享。

04 / Evaluation

批判性评价

从方法论、数据可信度、逻辑严密性等维度输出批判性分析，辅助形成独立学术判断。

05 / Search

论文联合检索

聚合 Semantic Scholar 与 arXiv 双源检索，AI 自动理解并改写查询词，精准命中学术目标。

06 / Q&A

多轮上下文追问

上传即建立文档会话，支持携带历史的流式追问，上下文严格绑定当前文档，不跨文档污染。

How it works

四步完成论文精读

01

上传文档

支持 PDF、DOCX、PPTX、TXT，最大 16 MB。拖拽或点击上传，无需注册账号。

02

流式分析

SSE 实时推送各模块结果，摘要、引用、结构图并发生成，无需等待全部完成。

03

检索延伸

基于当前论文内容自动生成检索主题，推荐延伸阅读，一键导入并替换分析会话。

04

导出沉淀

导出完整分析报告（Markdown）与 Mermaid SVG，阅读笔记随时带走。

Supported Formats

主流学术文档格式，全部支持

文本层提取，图表内文字自动识别。DOCX 含表格，PPTX 含备注页。

REST + SSE API

完整的工程化接口

所有功能均通过 FastAPI 暴露为标准接口。分析与追问同时支持非流式和 SSE 流式两种模式，可轻松集成到任意工作流。

POST/api/analyze/stream
POST/api/ask/stream
POST/api/search-papers
POST/api/recommend-papers
GET/api/download-paper
